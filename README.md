# Remote Staffing Backend Platform

## Overview
A backend system for a marketplace-style staffing platform that enables job listings, candidate ingestion, matching, and scoring using a scalable serverless architecture.

## Features
- Job description ingestion and processing
- Candidate upload and profile handling
- Matching and scoring workflows
- RESTful APIs for platform operations
- Event-driven and serverless backend design

## Tech Stack
- Backend: Python, FastAPI
- Cloud: AWS Lambda, API Gateway, S3, DynamoDB
- Databases: DynamoDB
- Tools: Git, Postman

## Architecture
- Serverless microservices using AWS Lambda
- API Gateway for routing
- Modular services for ingestion, scoring, and matching

## Relevance
This project demonstrates experience in building scalable backend systems for listing-based platforms, similar to marketplaces and booking systems.
